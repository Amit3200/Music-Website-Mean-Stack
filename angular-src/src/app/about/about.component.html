Zyrix Music



Zyrix Music Website is a project built on MEAN stack. Napster api allows the user to fetch the musics and other details. The user can pin the song as its favourite and can go through the napsters top music play lists. This application allows user to get the list of trending musics all over the globle. User has to login to access its playlist and can play the songs in the anonymous mode.
